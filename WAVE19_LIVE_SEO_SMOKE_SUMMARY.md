# Wave 19 — Live SEO Smoke Readiness

This wave adds a lightweight live verification layer to use immediately after deployment.

## Added

- `data/live_verification_targets.json`
- `scripts/live_seo_smoke_audit.py`
- `GO_LIVE_COMMANDS.md`
- `LIVE_SEO_SMOKE_AUDIT.md` is generated by the script

## Purpose

After Vercel deployment, run one command to check priority routes, basic metadata, and obvious old-branding regressions.

```bash
RUN_LIVE_CHECKS=true BASE_URL=https://bharathqr.com npm run live-seo-smoke
```

The script is offline-safe by default, so it will not break CI or prelaunch checks when the site is not live yet.

## Build philosophy

No new product scope was added. This is a verification and launch-confidence wave only.
