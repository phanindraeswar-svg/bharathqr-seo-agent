# Homepage Positioning Audit

Positioning locked:

**BharathQR is India’s AI-Powered QR & Business Growth Platform.**

Homepage should make three things obvious within the first screen:

1. Generate useful QR codes for Indian businesses.
2. Use tools for payments, reviews, WhatsApp, WiFi, menu, events, and business contacts.
3. No login, no database, no hidden fees for static QR tools.

Recommended above-the-fold language:

> Create free QR codes for UPI payments, Google reviews, WhatsApp chats, WiFi sharing, menus and business cards. Built for Indian shops, restaurants, hotels, clinics, salons and small businesses.

Homepage internal-link priorities:

- Tools directory
- Solutions directory
- Use Cases directory
- Trust/Safety pages
- Cases hub

Post-launch check:

- If homepage impressions are high but CTR is low, test simpler title/meta copy focused on "Free QR Code Generator for Indian Businesses".
